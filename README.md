# MJL-1 Calculator

A clean, modern calculator built with vanilla HTML, CSS, and JavaScript as part of The Odin Project Foundations curriculum.

## Live Preview

[View Live](https://malvin149.github.io/js-calculator)

## Features

- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Decimal point support with duplicate decimal guard
- Sign toggle (+/-) for positive/negative numbers
- Double zero (00) input with leading zero protection
- Delete/backspace button to remove last entered digit
- Divide by zero protection with error handling
- Long decimal rounding using toPrecision
- Dynamic font scaling for long numbers
- Keyboard support
- Light mode UI with MJL-1 branding

## Technologies

- HTML5
- CSS3 (Flexbox, CSS Custom Properties, Google Fonts)
- Vanilla JavaScript (ES6+)

## How to Use

1. Clone the repository
2. Open `index.html` in your browser
3. Use mouse clicks or keyboard to operate the calculator

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0-9` | Digits |
| `+` `-` `*` `/` | Operators |
| `Enter` | Equals |
| `Backspace` | Delete last digit |
| `Escape` | Clear |
| `.` | Decimal point |
